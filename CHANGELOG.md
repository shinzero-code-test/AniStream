# CHANGELOG

## [2.0.1] - 2026-04-28

### إصلاحات
- إصلاح استدعاءات Compose داخل أقسام الصفحة الرئيسية لضمان نجاح التجميع.
- إصلاح الحاوية الجذرية للتنقل السفلي وإزالة الاعتماد غير الصحيح على `calculateBottomPadding`.
- استكمال وسم `FlowRow` التجريبي في بطاقة تفاصيل العمل لضمان مرور البناء متعدد المعماريات.

## [2.0.0] - 2026-04-28

### جديد
- إضافة شريط تنقل سفلي مع صفحات مستقلة للرئيسية والمكتبة والإعدادات.
- إضافة شاشة مكتبة مخصصة تضم تبويباً للقائمة وتبويباً للسجل.
- إضافة حالات مشاهدة محلية مستوحاة من Anime3rb: أشاهده حالياً، أنهيت مشاهدته، سأكمله لاحقاً، لن أكمله، سأشاهده لاحقاً.
- إضافة تقييم شخصي محلي من 1 إلى 10 داخل صفحة العمل.
- إضافة قسم "أكمل المشاهدة" في الصفحة الرئيسية اعتماداً على سجل التشغيل المحلي.
- إضافة الترتيب المباشر للفهرس والبحث مع التحميل التدريجي للصفحات من Anime3rb.com.
- إضافة دعم مباشر للعروض التشويقية، الروابط الخارجية، والأنميات المشابهة المقترحة من صفحة العمل.
- إضافة معلومات إضافية في شاشة المشغل مثل عدد المشاهدات، رابط تحميل جميع الحلقات، والانتقال للحلقة التالية.
- إضافة إعدادات محلية لحفظ تفضيلات التشغيل التلقائي، الملخص، الوضع السينمائي، والألوان الديناميكية.
- إضافة دعم عربي شامل لواجهات التطبيق مع اتجاه RTL.

### تحسينات تقنية
- الاستمرار في جلب البيانات مباشرة من Anime3rb.com مع استخدام ملفات HTML المحلية فقط للتحقق من البنية والمحددات.
- توسيع طبقات Clean Architecture وRoom/DataStore لدعم الميزات الجديدة.
- تفعيل بناء APK لكل المعماريات الرئيسية: `armeabi-v7a`, `arm64-v8a`, `x86`, `x86_64` مع نسخة `universal`.
- أتمتة إنشاء إصدار GitHub Release بعد كل بناء ناجح على الفرع `main` مع إرفاق ملفات APK وCHANGELOG.

## [1.0.0] - 2026-04-28

### الأساس
- تهيئة مشروع AniStream باستخدام Kotlin وJetpack Compose وMVVM + Clean Architecture.
- إضافة محرك كشط أولي يعتمد على OkHttp وJsoup وMedia3 مع دعم Cloudflare عبر WebView.
- إضافة قاعدة بيانات محلية للقائمة والسجل، مع إعدادات DataStore أولية.
