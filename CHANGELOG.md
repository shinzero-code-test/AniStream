# CHANGELOG

## [2.2.0] - 2026-04-29

### جديد
- توليد مخططات واجهات Premium جديدة عبر Google Stitch لمشهد الصفحة الرئيسية ومشهد المشغل/التفاصيل مع توثيقها في `docs/STITCH_UI_SCHEMAS.md`.
- إضافة منطق `AutoFailoverPlaybackResolver` لتدوير مصادر التشغيل البديلة وإعادة استخراج الروابط المؤقتة عند فشل المصدر الحالي.
- إضافة خدمة `AnimeDownloadService` لتنزيل الحلقات في الخلفية وتحويل بث HLS إلى ملف محلي مع إشعار foreground وتقدّم حي.
- دعم إعادة استخراج رابط التشغيل عند أخطاء المشغل قبل الانتقال إلى خادم بديل.
- إضافة زر "تخطي المقدمة 85ث" داخل شاشة المشغل كأساس لميزة Smart Resume & Skip Intro.
- توسيع فلاتر الفهرس من جهة البيانات لدعم `season`, `year`, `genres`, `status`, و `age` في طلبات Anime3rb المباشرة.

### تحسينات
- إضافة دعم أولي لملفات `MKV` ضمن مكتشف المصادر.
- تفعيل Picture-in-Picture على مستوى الـ Activity تمهيداً لتجربة تشغيل متميزة.

## [2.1.1] - 2026-04-29

### إصلاحات
- إصلاح تعطل `CloudflareWebViewInterceptor` عند انتهاء المهلة أو إلغاء المهمة، عبر ضمان تنفيذ تنظيف `WebView` دائماً على خيط الواجهة الرئيسي.
- جعل تنظيف `WebView` آمناً ومتكرراً مرة واحدة فقط لتفادي استدعاء دوال WebView من `DefaultExecutor` الخاص بالكوروتينات.

## [2.1.0] - 2026-04-28

### جديد
- إضافة مسار `CloudflareWebViewInterceptor` كحلقة تمهيد صامتة عبر WebView لاستخراج الكوكيز وUser-Agent الحي قبل طلبات الكشط المباشر.
- ربط `OkHttp` بجلسة WebView الحية عبر `WebSessionStore` بحيث تستخدم طلبات Jsoup نفس الـ User-Agent النشط والكوكيز المكتسبة.
- تفعيل تمهيد الجلسة الحية قبل جلب الصفحة الرئيسية والبحث والتفاصيل وصفحات الحلقات من `anime3rb.com`.

### تحسينات
- إبقاء استخراج روابط الفيديو لحظياً فقط عند تشغيل الحلقة مع منع تخزين روابط البث المؤقتة.
- تحديث وثائق المشروع لتوضيح طبقة الجلسة الحية ومسار Cloudflare الجديد.

## [2.0.1] - 2026-04-28

### إصلاحات
- إصلاح استدعاءات Compose داخل أقسام الصفحة الرئيسية لضمان نجاح التجميع.
- إصلاح الحاوية الجذرية للتنقل السفلي وإزالة الاعتماد غير الصحيح على `calculateBottomPadding`.
- استكمال وسم `FlowRow` التجريبي في بطاقة تفاصيل العمل لضمان مرور البناء متعدد المعماريات.

## [2.0.0] - 2026-04-28

### جديد
- إضافة شريط تنقل سفلي مع صفحات مستقلة للرئيسية والمكتبة والإعدادات.
- إضافة شاشة مكتبة مخصصة تضم تبويباً للقائمة وتبويباً للسجل.
- إضافة حالات مشاهدة محلية مستوحاة من Anime3rb: أشاهده حالياً، أنهيت مشاهدته، سأكمله لاحقاً، لن أكمله، سأشاهده لاحقاً.
- إضافة تقييم شخصي محلي من 1 إلى 10 داخل صفحة العمل.
- إضافة قسم "أكمل المشاهدة" في الصفحة الرئيسية اعتماداً على سجل التشغيل المحلي.
- إضافة الترتيب المباشر للفهرس والبحث مع التحميل التدريجي للصفحات من Anime3rb.com.
- إضافة دعم مباشر للعروض التشويقية، الروابط الخارجية، والأنميات المشابهة المقترحة من صفحة العمل.
- إضافة معلومات إضافية في شاشة المشغل مثل عدد المشاهدات، رابط تحميل جميع الحلقات، والانتقال للحلقة التالية.
- إضافة إعدادات محلية لحفظ تفضيلات التشغيل التلقائي، الملخص، الوضع السينمائي، والألوان الديناميكية.
- إضافة دعم عربي شامل لواجهات التطبيق مع اتجاه RTL.

### تحسينات تقنية
- الاستمرار في جلب البيانات مباشرة من Anime3rb.com مع استخدام ملفات HTML المحلية فقط للتحقق من البنية والمحددات.
- توسيع طبقات Clean Architecture وRoom/DataStore لدعم الميزات الجديدة.
- تفعيل بناء APK لكل المعماريات الرئيسية: `armeabi-v7a`, `arm64-v8a`, `x86`, `x86_64` مع نسخة `universal`.
- أتمتة إنشاء إصدار GitHub Release بعد كل بناء ناجح على الفرع `main` مع إرفاق ملفات APK وCHANGELOG.

## [1.0.0] - 2026-04-28

### الأساس
- تهيئة مشروع AniStream باستخدام Kotlin وJetpack Compose وMVVM + Clean Architecture.
- إضافة محرك كشط أولي يعتمد على OkHttp وJsoup وMedia3 مع دعم Cloudflare عبر WebView.
- إضافة قاعدة بيانات محلية للقائمة والسجل، مع إعدادات DataStore أولية.
